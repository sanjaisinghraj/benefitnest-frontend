## Interactive Features

### Corporate Selection
- **Grid Layout**: Corporates displayed as selectable cards
- **Hover Effect**: Card highlights on hover
- **Active State**: Selected corporate shows blue border and light background
- **Info Display**: Corporate name, subdomain, and ID shown on each card

### Form Inputs
- **Color Input**: Dual input (color picker + hex text box)
- **Text Input**: Standard text field for text values
- **Number Input**: Spinner control for numeric values
- **Textarea**: Multi-line input for descriptions and CSS
- **Checkbox**: Toggle switches for boolean settings

### Toast Notifications
- **Success**: Green notification when saved
- **Error**: Red notification on failure
- **Position**: Fixed top-right corner
- **Auto-dismiss**: Disappears after 3 seconds

### Save Button
- **Disabled State**: When no corporate selected
- **Loading State**: Shows spinner while saving
- **Position**: Fixed top-right in header
- **Color**: Green (success variant)

## Form Validation

- **No Save Without Selection**: Save button disabled until corporate selected
- **Error Handling**: Network errors show toast
- **Success Feedback**: Confirmation toast on save

## Responsive Design

- **Grid Layout**: Auto-fit columns responsive
- **Card Grid**: Adapts to screen width
- **Form Fields**: Stack vertically on small screens
- **Mobile Friendly**: Scrollable sections on narrow viewports

## Accessibility

- **Semantic HTML**: Proper input labels and structure
- **Color Contrast**: WCAG AA compliant colors
- **Keyboard Navigation**: Tab through form fields
- **Screen Reader Support**: Proper label associations

## Performance

- **Lazy Load**: Customizations fetched only on corporate selection
- **Efficient Updates**: Only changed field sent on save (full object sent)
- **Debounced**: No automatic saves, manual control
- **Network**: Uses axios with proper error handling
